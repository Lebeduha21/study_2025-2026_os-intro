Теоретическое введение

Markdown — это легкий язык разметки, созданный для форматирования простого текста с помощью интуитивно понятных символов (звездочки, решетки, дефисы) без использования сложных HTML-тегов. Он позволяет быстро создавать заголовки, списки, ссылки и выделять текст, сохраняя исходный файл читабельным. Файлы имеют расширение .md
